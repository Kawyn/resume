#łańcuchy markova jako generator hrabiego monte christo

Nie pamiętam już, dlaczego stworzyłem owy projekt, ale Najprawdopodobniej ma to coś wspólnego z Hrabią Monte Christo. Opiera się on na łańcuchach markova, które są wykorzystane do generowania tekstu, czyli bierzemy tekst, i patrzyby na literkę/wyraz, poatrzymy na kolejny wyraz/literkę i mamy przejścia stanów. Potem gdy chcemy wygenerować tekst z "nauczonego" łańcucha wybieramy stan początkowy i "rzucamy" kością, następnie przechodzimy na wybrany stan i znów "rzucamy" kością. itp. Wspominałem, że nauka jest WiElOwĄtKoWa!?

W wymaganicha był jakiś kod, ale naprawdę nie ma czym się tam chwalić, także, tutaj jakiś losowy kod z projektu. 

                
for (let i = 0; i <= input.length - message.data.length; i++) {

    let depth = 0;
    let c = chain, w = input[i + depth]; 

    while (depth !== message.data.length - 1) {

        if (!c[w])
            c[w] = {};

        c = c[w];

        depth++;

        w = input[i + depth];
    }

    if (!c[w])
        c[w] = 0;

    c[w]++;
}
